NumberCamera

Приложение делает фото через системную камеру Android и сохраняет файлы как 001.jpg, 002.jpg, 003.jpg...

Как собрать APK через GitHub:
1. Загрузите все файлы в корень репозитория.
2. Откройте Actions.
3. Откройте Build APK.
4. Нажмите Run workflow.
5. После завершения откройте последний запуск и скачайте Artifact: NumberCamera-debug-apk.
