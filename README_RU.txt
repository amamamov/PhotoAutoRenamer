Photo Auto Renamer

Что делает:
- Следит за папкой DCIM/Camera.
- Новые фото переименовывает в 000001.jpg, 000002.jpg и так далее.
- Работает в фоне.

Как собрать APK через GitHub:
1. Распакуй этот архив.
2. Загрузи содержимое архива в репозиторий GitHub.
3. Открой вкладку Actions.
4. Выбери Build APK.
5. Нажми Run workflow.
6. После сборки скачай Artifact: PhotoAutoRenamer-debug-apk.

После установки:
1. Открой приложение.
2. Нажми «Дать доступ к файлам».
3. Разреши полный доступ к файлам.
4. Нажми «Запустить переименование».

Для Huawei/EMUI:
Настройки > Батарея > Запуск приложений > Photo Auto Renamer > Управлять вручную.
Включи: автозапуск, вторичный запуск, работу в фоне.
